
# blog to thread agentic-workflow

This project leverages CrewAI Flow to scrape a website, prepare a social post and publish it, powered by a locally running deepseek-R!!

## Use Cases

1. **Content Repurposing**: Automatically convert blog posts into engaging Twitter threads or LinkedIn carousels, maximizing content reach across different platforms.

2. **Social Media Automation**: Schedule and publish social media content derived from your website updates, maintaining consistent social presence without manual effort.

3. **Thought Leadership Sharing**: Transform long-form articles into digestible thread formats for social media, making complex ideas accessible to wider audiences.

4. **Content Marketing Pipeline**: Create an automated workflow from blog publication to social media distribution, ensuring every piece of content gets multi-channel exposure.


## Installation and setup

**Get API Keys**:
   - [FireCrawl](https://docs.firecrawl.dev/introduction)
   - [Typefully](https://support.typefully.com/en/articles/8718287-typefully-api)

   Run each code block in notebook.ipynb to run the project


**Install Dependencies**:
   Ensure you have Python 3.11 or later installed.
   ```bash
   pip install crewai crewai-tools
   ```
